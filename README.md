# 経費精算アプリ

このリポジトリは Google Apps Script ベースの経費精算アプリの正本です。

## 運用ルール
- 正本は GitHub
- 編集はローカルPC
- Apps Script へは clasp push で反映
- Apps Script を直接修正した場合は、すぐに clasp pull して GitHub に戻す

- Code.js
- Auth.js
- Drive.js
- OCR.js
- Category.js
- Sheets.js
- Report.js
- ListSheet.js
